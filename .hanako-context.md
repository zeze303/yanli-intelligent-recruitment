## 架构

一个 Express Web Service 同域名服务三个入口：
- `/` → 招生工具（查分选专业）
- `/helper` → 校园助手（AI 问答）
- `/api/chat` → AI 后端（DeepSeek V4 Flash）

- 数据：data.js（UMD 双端兼容，build-data.js 从 Excel 生成）

- 校园助手尚无代码

## 关键决策
- 合并为一个 Express Web Service，消除跨域
- MAJORS 对象格式（含 subject/phy_rank/his_rank）
- data.js UMD 格式，浏览器/Node 双兼容
- BATCH_LINE 由构建脚本生成，不硬编码
- AI 四层边界：字数(500)→敏感词→向量检索→prompt
- Express static 顺序：/helper 在前，/ 在后
- AI 推荐：前端传递 matchList，后端不加载全量 MAJORS
- render.yaml + package.json 已定义
