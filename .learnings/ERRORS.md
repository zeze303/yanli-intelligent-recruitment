## 2026-07-01: 本地 smoke test 默认占用 3000 端口失败
- **分类**: integration
- **描述**: 用 `Start-Job` 启动 `node server.js` 做接口 smoke test 时，3000 端口已被其他进程占用，导致 `EADDRINUSE`。
- **原因**: 测试脚本默认使用项目的默认端口，没有先探测端口是否空闲。
- **修正**: 本地验证时优先使用临时端口（如 3011）启动服务，避免和现有实例冲突。

## 2026-07-01: 误假设项目是 Git 仓库
- **分类**: integration
- **描述**: 尝试执行 `git status --short` 检查改动时失败，目录不是 Git 仓库。
- **原因**: 未先确认项目是否在版本控制下。
- **修正**: 运行 Git 命令前先确认 `.git` 是否存在，非仓库目录直接跳过 Git 状态检查。
