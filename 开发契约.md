## 一、data.js（UMD 双端兼容）

文件位置：`public/zhaosheng/data.js`

对外暴露四个全局变量（浏览器）/ 四个导出属性（Node）：

```javascript
MAJORS      // Array<MajorObject>  专业数据
RANK_2025   // Record<string, [4]> 2025一分一档
RANK_2026   // Record<string, [4]> 2026一分一档
BATCH_LINE  // BatchLineConst      省控线
```

### 1.1 MajorObject

```javascript
{
  name:       string,    // 专业名称，如 "计算机科学与技术"
  level:      '本科'|'专科',
  subject:    'wl'|'ls'|'all',   // 仅物理 | 仅历史 | 文理兼收
  score:      number,    // 冗余=phy_score
  rank:       number,    // 冗余=phy_rank
  phy_score:  number,    // 物理类投档分，仅历史专业为null
  phy_rank:   number,    // 物理类位次
  his_score:  number|null,
  his_rank:   number|null,
  desc:       string     // 专业描述
}
```

### 1.2 RankObject

```javascript
{
  "750": [15, 15, null, null],    // [phy_cnt, phy_cum, his_cnt, his_cum]
  "749": [8,  23,  null, null],
  ...
}
```

- key 统一为字符串（`String(score)`）
- cumIdx: 物理=1, 历史=3

### 1.3 BATCH_LINE

```javascript
{
  "2025": { "wl": 477, "ls": 459, "all": 200 },
  "2026": { "wl": 443, "ls": 485, "all": 200 }
}
```

## 三、API: POST /api/chat

```
Content-Type: application/json
Rate-Limit: 20 req/min per IP
```

### 3.1 校园问答模式

**Request:**
```json
{
  "source": "yanli",
  "question": "南院宿舍几人间？"
}
```

**Response 200:**
```json
{
  "answer": "南院宿舍为6人间，配备空调和独立卫生间..."
}
```

**Response 4xx:**
```json
{ "error": "参数校验失败" }
{ "error": "请求太频繁，请稍后再试" }
```

### 3.2 专业推荐模式

**Request:**
```json
{
  "source": "admissions",
  "score": 470,
  "rank": 202675,
  "subject": "wl",
  "matchList": [
    { "name": "土木工程", "tag": "保", "rank": 224000, "score": 455 },
    { "name": "建筑学", "tag": "稳", "rank": 218000, "score": 460 },
    { "name": "环境工程", "tag": "冲", "rank": 211000, "score": 465 }
  ]
}
```

**Response 200:**
```json
{
  "recommendations": [
    {
      "name": "建筑学",
      "reason": "你的位次（202,675）比去年录取位次（218,000）高约15,000名..."
    }
  ]
}
```

### 3.3 校验规则

```javascript
source 白名单：['yanli', 'admissions']
yanli:      question.length > 0
admissions: score ∈ [0,750], rank > 0, subject ∈ ['wl','ls'], matchList 非空
```


```
  GET  /                 → public/zhaosheng/index.html
```
